# TANKOLD APP
Esta es mi primera aplicación professional que hice cuando trabajaba en una empresa muy pequeña llamada TANKOLD por 10 meses, esta aplicación es un sistema IoT y Bluetooth que comunica un ESP32 por bluetooth a un teléfono android y este se comunica a un servidor MQTT, la aplicacion no esta terminada ni la voy a terminar, hacer una app con funciones complejas como Bluetooth low energy y mqtt con react native es muy impractico, la volveria a hacer pero en nativo con android studio porque el desarrollo con micro controladores, IoT y  domótica me apasiona mucho.   

Dejo este proyecto aquí porque aunque no lo termine, me gusto mucho poder crear un sistema de automatización que ayudara a la gente a hacer mas fail su vida.
También aprendí muchísimo con este proyecto, entendí que para aprender de verdad no vasta con clases normales de programación, videos de YouTube o cursos, la mejor forma de aprender es trabajar en un proyecto real.
Aprendí mucho sobre desarrollo real de aplicaciones, control de versiones con git, en mis primeros meses usaba un archivo .txt para documentar todo y clonaba mi directorio de trabajo haciendo que todas las copias pesaran mas de 116GB y luego descubri que git solucionaba ese problema , aprendi mucho sobre desarrollo de codigo para hardware, a diseñar PCBs desde cero y muchas cosas mas.
Espero seguir desarrollando muchos proyectos mas para seguir aprendiendo mucho mas.
